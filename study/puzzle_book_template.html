How to play

Larissa made this latin square, where each digit appears once per row and column. Quentin says “I made a latin square too! Let me color the positions our squares have in common”. What is Quentin’s square?

To solve Sudodles, you must use logical deduction to replace all the grey digits while ensuring that each digit is present exactly once per row and column.

The 400 puzzles in this collection are organized by grid size and level. You might also like the Sudodle web app at sudodle.app, which features a thousand more puzzles, and a multi-turn mode.
